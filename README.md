# 🤖 AI-Powered College Assistant Chatbot

## 📌 Description

The College Assistant Chatbot is an AI-powered chatbot developed using Python, NLP, and Streamlit to automate student query handling.

The chatbot answers questions related to:
- Admissions
- Attendance
- Exams
- Fees
- Departments
- Hostel
- Library
- Placement
- Sports

---

## 🚀 Features

- NLP-based chatbot
- Automated responses
- Chat history with timestamps
- Interactive Streamlit UI
- AI-powered query handling

## ▶️ Run Project

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py --server.port 8502
```
